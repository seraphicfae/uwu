#### Dotfiles for Arch Linux + Hyprland

You can find them <u>[here](https://github.com/seraphicfae/hyprland-dotfiles)</u> to use anytime!

### Features

- **Window Manager:** Hyprland  
- **Taskbar:** Waybar  
- **App Launcher:** Rofi  

This setup has a robust ecosystem.  
Everything is easily configurable with plenty of QoL features!
